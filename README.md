# Instagram Performance Dashboard – Power BI

This project analyzes Instagram performance using Power BI.

## Key Metrics

* Total Posts: 199
* Total Likes: 181K
* Total Comments: 32K
* Total Shares: 22K
* Total Saves: 26K
* New Followers: 8K
* Engagement Rate: 19.9%

## Dashboard Insights

* Monthly likes trend
* Monthly follower growth
* Engagement by content type (Reels, Carousel, Images)
* Top countries by follower growth
* Audience filtering by country and age group

## Tools Used

* Power BI
* Excel
* Data Visualization
